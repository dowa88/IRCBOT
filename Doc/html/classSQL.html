Detailed Description

Diese Klasse ist eine Datenbank.

Hier bei handelt es sich um einen Datenbank Klasse die mit SQLite arbeitet. Die Tabellen der Datenbank werden immer nach dem gleichen Schema erzeugt. Jede Tabele hat ein Splate name, Inhalt, Date und Uhr. Wobei Date und Uhr automatisch gefühlt beim erstellen mit Inhalt gefühlt werden.
